## 🚀 Getting Started (5-Second Handoff)

1. Open a PowerShell terminal.
2. Run `.\run_mobile.ps1`.
3. Verify the window is **400×800** and the **Bottom Tab Bar** is active.
4. Happy coding! 🕹️
