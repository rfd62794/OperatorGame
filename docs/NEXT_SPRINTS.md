# Future Sprints Roadmap

Following the successful completion of Sprint G.5 (Multi-Target Missions & Reliability), the following roadmap has been established for future development.

## Option A: G.6 Breeding System (Selected)
**Scope**: Genetics → Offspring → Trait Inheritance
**Focus**: 
- Implement `GeneticProfile` and trait inheritance logic.
- Add "Breeding" UI section in the Manifest/Roster tab.
- Introduce breeding cooldowns and resource costs (Scrap, XP).
- **Goal**: Expand roster depth and long-term progression via T3 legendary traits.

## Option B: G.7 Equipment Expansion
**Scope**: Gear System → Slots beyond Hats
**Focus**:
- Expand the `Equipment` model to include `Chest`, `Hands`, and `Feet` slots.
- Add dedicated shop categories in the Quartermaster.
- Implement specialized stat bonuses for high-tier gear.
- **Goal**: Deepen the tactical customisation and economy Sink.

## Option C: G.8 Advanced Injuries
**Scope**: Injury → Recovery Mechanics → Downtime
**Focus**:
- Transition to a more detailed injury model with varying recovery times.
- Implement "Medical Bay" or intensive care mechanics.
- Add risk-management layers to gauntlet missions (retreating vs. pushing).
- **Goal**: Increase the stakes of high-level operations.

**Current Selection: G.6 Breeding System**
